# blockchain

Proof of Work Blockchain implementation in TypeScript.
Exposed interface through an HTTP API.

## Use

### Prepare

`yarn`

### Start

`yarn start`

### Test

`yarn test`

## API

* `POST /api/transactions/new`
* `POST /api/nodes/register`
* `GET /api/nodes/resolve`
* `GET /api/mine`
